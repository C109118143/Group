# 內容
有一台飛機會自動飛行和發射子彈，玩家可控制飛行左右前後，利用飛行移動攻擊外星怪物，
怪物會掉落金錢或升級裝備，升級裝備可以是飛機的子彈發生變化，變得更強力於通關，也有可能調出防護罩之類的。
